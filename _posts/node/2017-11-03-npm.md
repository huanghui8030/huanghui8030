npm在国内用起来会比较慢，使用cnpm可完成替代npm，同时速度比较快。

cnpm为淘宝镜像的方法。

用cnpm替代npm即可，mac下终端执行如下代码，Windows下去掉sudo：

```js
sudo npm install cnpm -g --registry=https://registry.npm.taobao.org
```
