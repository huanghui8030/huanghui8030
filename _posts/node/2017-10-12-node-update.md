nodeJs大家经常会用到，但是更新太快，有些内容如果不更新会到时无法使用。例如antDesign如果使用的话，需要node版本大于0.6。更新方法，需要借助一个n这个模块，专门用来管理nodejs版本的。

方法如下：

- 全局安装，命令如下：`sudo npm install -g n`
- nodejs版本升级：
  - n stable 升级为文档版本
  - n latest 升级为官网最新版本
